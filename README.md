# Agent Ops

This folder stores reusable orchestration assets for future agent/tool workflows.

## Structure
- `workflow.yml` - canonical workflow policy
- `tasks/todo.md` - execution planning template
- `tasks/lessons.md` - correction-driven learning log
- `agents/` - future agent profiles and runbooks
- `tools/` - future tool-specific guides/config
- `skills/` - future skill playbooks and patterns

## Usage
1. Copy/update `tasks/todo.md` for each task.
2. Execute work while checking off items.
3. Add verification evidence in `todo.md`.
4. On any correction, append an entry in `tasks/lessons.md`.
